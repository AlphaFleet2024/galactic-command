# Security Policy

Galactic Command is committed to ensuring the security of our game and its players. We value and encourage community involvement in identifying vulnerabilities and areas for improvement.

## Supported Versions

| Version   | Supported         |
| --------- | ----------------- |
| 1.0.x     | :white_check_mark: |
| < 1.0.0   | :x:               |

## Reporting a Vulnerability

If you discover a security vulnerability, please help us address it responsibly:

1. **Do not disclose it publicly.** Instead, please email our security team at `security@galacticcommand.com`.
2. Provide as much detail as possible about the vulnerability and the steps to reproduce it.
3. Our team will respond within 7 days with an initial assessment and any further questions.

## Security Best Practices

We follow industry standards to secure our game, including:

- Regular updates to dependencies to fix known vulnerabilities.
- Following best practices for authentication and data handling.
- Strict access control and database protection.

Thank you for helping keep Galactic Command safe for all players!
